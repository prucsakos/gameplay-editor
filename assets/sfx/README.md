# Sound Effects Library

Place short (1-5 second), royalty-free sound effect files here in MP3 format.

## Bundled SFX (add these manually)

The following files are expected by the funny style. Source them from royalty-free
libraries (freesound.org, pixabay.com/sound-effects, etc.):

- `bruh.mp3` — "bruh" meme sound
- `sad_violin.mp3` — sad violin sting for fails
- `fail_horn.mp3` — game show fail horn
- `laugh_track.mp3` — short laugh track
- `dramatic_boom.mp3` — dramatic impact/boom for comebacks
- `oof.mp3` — "oof" / Roblox death sound
- `airhorn.mp3` — MLG airhorn
- `vine_boom.mp3` — Vine boom sound effect

Requirements: each file should be <500KB, 1-5 seconds, MP3 format.

## User SFX

Users can also place a `sfx/` directory next to their source video.
User sounds with the same filename override bundled sounds.
